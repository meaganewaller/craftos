# Gauge Calculator

A web application for calculating knitting and crochet gauge. Given a swatch measurement, it computes stitches per inch (SPI), rows per inch (RPI), and the number of stitches or rows needed for a target dimension.

Built with [Sinatra](https://sinatrarb.com) and powered by the [fiber_units](../../gems/fiber_units) and [fiber_gauge](../../gems/fiber_gauge) gems from CraftOS.

## Getting Started

```sh
cd apps/gauge-calculator
bundle install
bundle exec rackup
```

The app starts at `http://localhost:9292`.

## How It Works

Enter your swatch measurements (stitch count, row count, and width) and the app calculates your gauge. You can then enter a target width or height to find out how many stitches or rows you need.

Units are supported in both inches and centimeters.

## API

All endpoints accept JSON request bodies and return JSON responses.

### `POST /api/gauge`

Calculate stitches per inch and rows per inch from a swatch.

**Request:**

```json
{
  "stitches": 20,
  "rows": 28,
  "width": 4,
  "unit": "inches"
}
```

**Response:**

```json
{
  "spi": 5.0,
  "rpi": 7.0
}
```

### `POST /api/gauge/stitches`

Calculate the number of stitches needed for a target width.

**Request:**

```json
{
  "stitches": 20,
  "rows": 28,
  "width": 4,
  "target_width": 38,
  "unit": "inches"
}
```

**Response:**

```json
{
  "stitches": 190
}
```

### `POST /api/gauge/rows`

Calculate the number of rows needed for a target height.

**Request:**

```json
{
  "stitches": 20,
  "rows": 28,
  "width": 4,
  "target_height": 10,
  "unit": "inches"
}
```

**Response:**

```json
{
  "rows": 70
}
```

The `unit` parameter is optional and defaults to `"inches"`. Pass `"centimeters"` to work in metric.
